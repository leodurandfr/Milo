PopularityRank
==============

Returns how popular the podcast is. Returns null if in none of the below categories.

```
enum PopularityRank {
  TOP_200 // is a top 200 podcast
  TOP_1000
  TOP_2000
  TOP_3000
  TOP_4000
  TOP_5000
  TOP_10000
  TOP_20000
  TOP_50000
  TOP_100000
  TOP_200000
}
```
