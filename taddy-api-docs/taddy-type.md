TaddyType
=========

The different types of media available on Taddy

```
enum TaddyType {
  PODCASTSERIES
  PODCASTEPISODE
	COMICSERIES
  COMICISSUE
	CREATOR
}
```


`PODCASTSERIES` - A podcast

`PODCASTEPISODE` - An episode of a podcast

`COMICSERIES` - A comic

`COMICISSUE` - An issue (episode) of a comic

`CREATOR` - A creator (that made the podcast or comic).
